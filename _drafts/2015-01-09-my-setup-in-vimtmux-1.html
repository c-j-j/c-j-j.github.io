Here is one of the windows I currently have fortmux to do Ruby development. Two panes to the left for unit test and production code, and one on the right for test output;

Before I describe my setup in vim/tmux, I do have the plugins vim-sensible and tmux-sensible to add some basic common settings.

I save my setup this using the plugin tmux-resurrect so that next time I come restart my laptop I can immediate continue with my work.

I've added the following options in the ~/.tmux.conf file;

1. enable mouse scrolling (useful for scrolling test output);

set -g mode-mouse on

2. change tmux prefix from control b to control-a as it's more comofrtable.

unbind C-b
set -g prefix C-a

3. Set up new windows and panes to open in same working directory as you are working on.

bind '"' split-window -c "#{pane_current_path}"
bind % split-window -h -c "#{pane_current_path}"
bind c new-window -c "#{pane_current_path}"
